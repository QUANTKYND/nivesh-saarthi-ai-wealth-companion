# MVP Notes

## Scope

This repository currently contains scaffolding only. Business features, integrations, AI workflows, portfolio logic, onboarding, recommendations, and analytics are intentionally deferred.

## Initial Technical Priorities

- Keep DTOs and API contracts in `packages/shared-types`.
- Add backend modules only when a real MVP workflow is selected.
- Add frontend routes and screens only after product scope is confirmed.
- Prefer typed API contracts and explicit environment configuration.

## Candidate Future Work

- Authentication and session handling.
- Investor profile capture.
- Portfolio summary APIs.
- Advisor review workflow.
- AI-assisted insight generation.
- Audit logging and compliance notes.
